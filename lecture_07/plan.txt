API VK
1. Разбираемся, что такое API
2. Создаем приложение в VK, оно нужно для доступа к аккаунту    https://vk.com/editapp?act=create
3. Устанавливаем модуль с VK API
4. Делаем авторизацию в системе с помощью файла с логином и паролем   15-20 минут
5. Открываем https://vk.com/dev.php?method=methods и смотрим, что там есть
6. Смотрим несколько методов API  25-30 минут
    users.get
    friends.get
    wall.post
    wall.get
    wall.repost
7. Проверяем парадокс дней рождения, записав дни рождения подписчиков какого-то паблика  25-30 минут
    выбрать несколько случайных элементов списка –– random.sample(l, n)
